Proyecto Calculadora:

-Necesidades:

	-Menú inicial con el que interactuar con la calculadora.
	
	-Posibilidad de realizar las operaciones básicas.

	-Posibilidad de realizar las mismas funciones que una calculadora científica

		  -Ayuda de funciones( https://hiraoka.com.pe/blog/post/calculadora-cientifica-que-es-y-para-que-sirve )

	-Posibilidad de ver los resultados y mostrarlos por pantalla.

	-Menú lateral derecho superior que muestre las operaciones realizadas y su respectivo resultado.

	-Menú lateral derecho inferior en el que seleccionar las diferentes opciones.

	-La calculadora funciona por paneles, 
	

-Layout: pantalla  ajustable dependiendo de la configuración y de si el programa se minimiza .

	-Lateral izquierdo => menú de opciones del programa, ( calculadora básica, calculadora 	científica ,historial, )
		
	-El menú de opciones abre los diferentes paneles con las dos posibles calculadoras 	existentes,la básica y la científica

   -El funcionamiento provisional se realizará con SetLayoutNull.

    -Se harán diferentes clases para lás operaciones, en las que cada método realizará una de las operaciones deseádas.
